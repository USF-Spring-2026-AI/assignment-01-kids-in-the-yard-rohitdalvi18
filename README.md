# AI Assignment 01 - Kids in the Yard

## Modules
- `main.py`: entry point, loads CSV data, generates the tree, runs the menu.
- `person.py`: defines the `Person` class (basic attributes + relationships).
- `person_factory.py`: reads CSV files and creates people (names, gendered-name logic, death year, last name rules).
- `family_tree.py`: builds the family tree (spouses + children rules) and provides `T/D/N/Q` queries.
- `reflection.txt`: responses for the Reflection section.
- `comparison.txt`: responses for the Comparison section (LLM prompt).


## Run
```bash
python3 main.py
